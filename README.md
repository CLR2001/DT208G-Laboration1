# ◈ Laboration 1
Detta repository innehåller laboration 1 för kursen DT208G.


## ✦ Projektbeskrivning
Denna webbplats tillåter användaren att spara kurser via ett formulär som sedan visas upp på en annan undersida.

## ⚙ Funktioner
* **TypeScript-kontroll:** Säkrare kod som skapar element utan krascher.
* **Inga omladdningar:** Smidig SPA-navigering.
* **Vite:** Automatiserad byggprocess med Vite.

## ⌨ Installation & Setup

1. **Klona projektet:**
   ```bash
   git clone https://github.com/CLR2001/DT208G-Laboration1.git
   ```
2. **Installera beroenden:**
   ```bash
   npm install
   ```
3. **Starta projektet:**
   ```bash
   npm run dev
   ```
   Eller
   ```bash
   npm run build
   npm run preview
   ```
